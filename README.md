## Why These Topics?

These three topics cover the majority of problems seen in coding interviews and competitive programming. Trees and Graphs deal with structural/relational data, while Dynamic Programming is a problem-solving technique that optimizes recursive solutions.

Mastering these gives a strong foundation to tackle most medium and hard level problems.

---

## Approach

- Understand the core concept before jumping into problems
- Start with easy problems, move to medium once comfortable
- Note down time and space complexities for every pattern learned
- Revisit problems that were solved with hints
